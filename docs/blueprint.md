# **App Name**: Insight Energy Governance Hub

## Core Features:

- User Authentication and Roles: Secure user authentication using Firebase Auth with customizable role-based access control (RBAC) for managing user permissions and access levels.
- Meeting Management: Schedule meetings, define agendas, manage participants, and integrate with calendar services (Google Calendar, Outlook) using .ics file generation. Allow file upload for supporting documents.
- Voting Module: Implement a voting module supporting open, secret, and proxy voting modes, weighted votes based on user roles, configurable quorum and majority rules, and automated result calculation.
- Automated Minute Generation: Utilize Gemini AI to process meeting transcriptions, generate executive summaries, structured minutes, and action plans with assigned tasks, responsibilities, and deadlines. This acts as a tool by summarizing discussions and decisions made in meetings.
- Project Portfolio Management: Manage project portfolios with details like name, code, status, responsible committee, and financial impact. Includes filtering, and integration with ERP Sankhya via REST API to synchronize financial data.
- Real-time Notifications: Multi-channel notification system (email, WhatsApp) for events like meeting invitations, voting reminders, and task assignments. Users can customize notification preferences.
- Interactive Dashboards and Reports: Generate customizable dashboards with charts, tables, and key indicators for projects, votes, and meeting participation. Provides filtering, and export options in PDF, CSV, JSON, and XLSX formats.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust, authority, and corporate stability.
- Background color: Very light lavender (#F0F2FA), near-white background, creates a clean and professional look.
- Accent color: Teal (#009688) for interactive elements and important CTAs to draw user attention without being overwhelming.
- Headline font: 'Poppins', a geometric sans-serif to maintain a modern, fashionable, avant-garde feel in headlines.
- Body font: 'PT Sans', a humanist sans-serif, will provide the text blocks a warmer more friendly feel that increases readability and helps new users to orient themselves in the app's interface.
- Use a consistent set of professional icons, with an outline style, related to governance, meetings, and project management.
- Maintain a clean and structured layout with a left sidebar for navigation and well-defined content sections, ensuring intuitive navigation.
- Incorporate subtle animations and transitions to provide feedback on user interactions, such as loading indicators and form submissions.
